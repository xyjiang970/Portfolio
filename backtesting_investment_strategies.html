SSL Error¶

Depending on the version of jupyter/ pandas you have, you may run into a SSL error if you use:

pd.read_html()

to fetch table data. To fix this, please refer to the link here: https://moreless.medium.com/how-to-fix-python-ssl-certificate-verify-failed-97772d9dd14c

In [2]:
# Fixing Python SSL Certificate Failed Error

import os, ssl

if (not os.environ.get('PYTHONHTTPSVERIFY', '') and
getattr(ssl, '_create_unverified_context', None)):
    ssl._create_default_https_context = ssl._create_unverified_context

Scraping DJIA From Wikipedia¶

"The DJIA includes only 30 large companies and is a price-weighted index, unlike later stock indices which use market capitalization." -Wikipedia

In this section we're going to grab all 30 tickers and append it to a list in order to grab historical data using yfinance later on. To keep things simple, instead of using beautiful soup, I decided to just use the built in pandas .read_html() method that is able to fetch data from tables. More info. about this method can be found in the official docs here: https://pandas.pydata.org/docs/reference/api/pandas.read_html.html

In [3]:
temp = pd.read_html("https://en.wikipedia.org/wiki/Dow_Jones_Industrial_Average")
# temp should be of type <class 'list'>

Unicode Normalization¶

Sometimes when using "pd.read_html( )" the scraped info. from tables contains unicode strings that cannot be seen when using the .info() method (it will just say the data types within the column are "objects"). We have to manually check ourselves by looking at the data - so in the example above we manually check the columns for unicode which would look something like (if looking at unicode data in the "Symbols" column):

"NYSE:\xa0MMM"
"NYSE:\xa0AXP"

Note: We see that we do not have any unicode strings in our scraped table from wikipedia, but this could be due to the recent jupyter notebooks & tools version I have when doing this project. Below is an image of all the current versions I'm using.

Next let's calculate the relative returns of DJI. Relative return is better here for comparisons to other similar investments while absolute return is just is what an asset or fund returned over a certain period of time (more meaningful to work with relative returns).

Relative return is also known as "alpha" (more on that here: https://www.investopedia.com/terms/a/alpha.asp).

We'll create a new column in our data frame that calculates the relative change (percent change) using the .pct_change( ) method (more info. on the method here: https://pandas.pydata.org/docs/reference/api/pandas.DataFrame.pct_change.html)

In [18]:
# Column for Return (relative change)
df['DJI_Return'] = df.pct_change()

# because we do not have the prices of previous timestamps.
df.dropna(inplace = True)
